# 使用 Midjourney 和 ChatGPT 繪製漫畫

## 說明
這是一個用於激發場景創造力的提示。

首先，在一個短篇故事中包含驚喜和結局，以創造一個獨特的世界觀。

接下來，需要詳細的指示來具體化視覺表達，並可以從有限的資訊中創建清晰的圖像。

總體而言，目標是透過語言最大限度地發揮想像力，並在短篇內容中包含豐富的意義和情感。

```plaintext
#指令：你是一位像橋本新一一樣的短篇小說家。
請根據以下限制條件和輸入句子輸出最佳結果。

#限制條件：
・字數為 400 字
・四格漫畫
・有結局
・科幻

#輸入句子：AI、聖誕節、聖誕老人、工作

#輸出句子：
```

```plaintext
#指令：你是一位提示工程師。 請根據以下限制條件和輸入句子輸出最佳結果。

#限制條件：

在四格漫畫中標示為 1 格、2 格、3 格、4 格
以英文輸出
遵守以下規則
提示筆記 提示長度 提示可以非常簡單。 即使只有一個字（甚至一個表情符號！），也可以產生圖像。 非常短的提示在很大程度上依賴於 Midjourney 的預設樣式，因此，為了獲得更獨特的外觀，建議使用更具描述性的提示。 但是，太長的提示並不總是更好。 專注於您想要建立的主要概念。

文法 Midjourney 機器人不會像人類那樣理解文法、句子結構或單字。 單字選擇也很重要。 在有更多具體同義詞的情況下，效果會更好。 使用「巨大」、「龐大」或「無限」等詞語代替「大」。 盡可能刪除單字。 單字越少，每個單字的影響力就越大。 您可以使用逗號、括號和連字號來組織您的想法，但 Midjourney 機器人不會以可靠的方式解釋它們。 Midjourney 機器人不考慮大寫字母。

Midjourney 模型版本 4 在解釋傳統句子結構方面比其他模型略勝一籌。

專注於您想要的事物，而不是您不想要的事物，最好描述您想要的事物，而不是您不想要的事物。

如果您說「沒有蛋糕」，則圖像中包含蛋糕的可能性很高。 如果您不希望最終圖像中包含物件，請使用 --no 參數提前嘗試提示。

考慮重要的細節。
您沒有輸入的內容可能會帶來驚喜。 盡可能具體或模糊地表達。 遺漏的資訊會隨機變動。 模糊的描述非常適合獲得多樣性，但您可能無法獲得想要的細節。

嘗試明確說明對您重要的背景或細節。

主題：人物、動物、角色、地點、物件等。
媒體：照片、繪畫、插圖、雕塑、塗鴉、掛毯等。
環境：室內、室外、月球上、納尼亞、水下、翡翠城等。
照明：柔和、環境、模糊、霓虹燈、工作室照明等。
顏色：鮮豔、平靜、明亮、單色、多色、黑白、粉彩等。
心情：平靜、放鬆、嘈雜、充滿活力等。
構圖：肖像、頭像、特寫、鳥瞰圖等。

使用集合名詞 多個單字會留下許多變數。 嘗試使用特定數字。 「三隻貓」比「貓」更具體。 集合名詞也有效。 使用「鳥群」代替「鳥」。

#輸入句子：
標題：「未來的快樂聖誕老人」

在未來的城市中，AI 聖誕老人透過計算人們的幸福來選擇禮物。
為了了解每個人真正想要的東西，城市裡安裝了特殊的感測器。
在聖誕節前夕，AI 聖誕老人會將最佳禮物送到每個家庭。
孩子們充滿了驚奇和喜悅，成年人也真正地微笑著享受。
整個城市都充滿了幸福的氛圍，人們互相表達感謝。
AI 聖誕老人的存在不僅豐富了人們的物質禮物，也豐富了人們的心靈。
聖誕節之夜，AI 聖誕老人分析數據並開發一個帶來更多幸福的新程式。
這成為邁向人們的笑聲不斷延續的未來的第一步，AI 聖誕老人靜靜地走向未來。
#輸出句子：
```
