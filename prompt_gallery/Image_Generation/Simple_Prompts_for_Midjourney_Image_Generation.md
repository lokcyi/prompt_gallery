# 用於 Midjourney 圖像產生的簡單提示

## 說明
這是一個提供圖像產生具體指示的提示。

您可以產生簡單的背景或動畫風格的插圖。

透過分別以日語和英語建立多種變體，您可以應對各種廣泛的表達方式或用途。

透過準備英語和日語指南，預計可以回應更廣泛的需求。

## 代碼塊

```plaintext
編寫背景圖像、簡單且純色背景的提示，並以日語和英語產生 10 個
```

