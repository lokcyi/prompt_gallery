# 輕鬆進行文件校對

## 說明
文件校對不僅要檢查語法或拼寫，文件給人的「印象」和「一致性」也同樣重要。

## 使用示例

以下代碼塊是要求文件校對的模板：

```plaintext
請校對以下文件。
文件的目標讀者是{預期讀者}，內容是關於{要傳達的內容}的說明。
{預期的文體風格}是理想的。請校對文件，並以表格形式說明修改的部分。

"""

(需要校對的文件) 

"""
```

此外，還有一個相同結構的其他示例：

