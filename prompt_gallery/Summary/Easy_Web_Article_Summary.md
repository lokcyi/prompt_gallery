# 輕鬆摘要網路文章

## 說明

此提示旨在整理特定網頁的資訊，並使其易於理解。 它通過內容描述和概述提供網頁的基本資訊，並闡明正文的目的和內容。

它包含“什麼”、“何時”、“何地”、“誰”、“如何”等具體資訊，最後說明與該網頁相關的問題。 通過這種方法，讀者可以在短時間內理解網頁的整體情況。

### 使用範例

請執行以下提示。

```plaintext
網頁的 URL：

[內容說明／概述]  
[正文的目的]  
[什麼]  
[何時？]  
[何地？]  
[誰？]  
[如何？]  
[問題點]
```
