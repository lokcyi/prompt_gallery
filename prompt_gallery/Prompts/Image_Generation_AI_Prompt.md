# 圖像生成 AI 提示

## 說明
這是一個提示，供專業插畫家向圖像生成 AI 運營商傳達設計理念。

可以根據製作要求創建最佳提示。句子簡潔，包含重要關鍵字，替換輸入句子中的單詞，並以情感方式表達。

提示的輸出將以英語和日語編寫，風格僅限於 pixiv、GALLERIA、deviantART 等流行的插畫 SNS 風格。此外，還必須描繪一個全身人物。
