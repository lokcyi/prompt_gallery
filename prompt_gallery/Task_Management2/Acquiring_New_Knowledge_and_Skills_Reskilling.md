# 學習新知識和技能的「技能重塑」

## 說明

這是一個提示，旨在為初學者提供有效的技能學習計畫。

透過詳細設定每日的學習內容和時間，並逐步提高難度，可以持續學習技能。

由於包含實際的專案，因此理論與實踐可以均衡學習，這是一大優點。

如果在研討會中使用，可以幫助參與者制定具體的學習行動計畫。

```plaintext
我想從零開始學習「知識和技能」。

請為初學者制定為期「天數」天的學習計畫。
計畫中請包含每日的學習內容、學習時間，以及像專案這樣的實踐要素。
請逐步提高難度，並確保能夠持續學習。
```
