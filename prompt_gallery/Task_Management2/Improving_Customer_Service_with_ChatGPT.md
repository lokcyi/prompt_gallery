# 用ChatGPT改進客戶服務

## 說明
這是一個用於生成客服人員撰寫常見問題(FAQ)、產品資訊、投訴處理、客戶調查所需範本的提示詞。

用中學生也能理解的方式來說明，就是幫助客服人員回答客戶經常問的問題、詳細解釋產品、處理問題發生時的應對方法等。

這個提示詞的特色是將客服工作中常見的溝通任務轉化為易於執行的步驟，讓新人也能快速掌握並提供專業的服務。

### 代碼塊示例

```plaintext
你是客服人員。
請生成回應客戶詢問的範本。
例如：
・產品退貨的方法是什麼？
・我想重設帳戶密碼。
```
