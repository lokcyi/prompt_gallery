**提示工程中有效的標籤**:  
1. `<instruction>`, `<task>`, `<steps>`等標籤可以清晰地定義任務指示和流程，在提示工程中非常有用。
2. `<summary>`, `<example>`, `<demo>`標籤可以提供摘要信息和示例，有助於設置更好的上下文。
3. `<condition>`, `<if>`, `<else>`等條件標籤在動態提示生成或條件響應中非常有用。
4. `<feedback>`和`<goal>`標籤可以幫助用戶設置目標並基於反饋進行優化。
