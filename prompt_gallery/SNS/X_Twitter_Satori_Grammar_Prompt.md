# 製作 X (Twitter) 悟道文法提示

## 說明
這是一個用於支援希望在 X (Twitter) 上提高影響力的個人或企業撰寫貼文的提示。

要求設定具體目標受眾，並使用與該受眾產生共鳴的「悟道文法」撰寫貼文。

最終目標是增加貼文的「喜歡」或轉發次數，並獲得新的追蹤者。

提示中設定了貼文主題或格式的決定、選擇適當的 Hashtag 等詳細條件。

# 悟道文法的特徵=
- 引導目標受眾的共鳴：透過提出具體問題，讓目標受眾感受到「這真的是我的故事！」並提高對產品的關注。
- 明確傳達產品的價值：提出具體的優點，而不是抽象的表達，讓使用者清楚了解產品如何提供幫助。
- 引導行動：提高對解決問題的期望，引導使用者嘗試使用產品。
- 吸引廣泛人群：展示不僅對目標受眾，而且對非目標受眾也有好處，從而吸引更多人的關注。
- 在工具的優點 1 或工具的優點 2 中僅使用一次「10 倍」這個詞。
