# 生成HTML草稿

## 說明
這是一個用於生成HTML格式表單的提示詞。根據特定的規格要求來生成所需的表單元素。

請根據條件限制和組成要求，

#條件限制：
・使用GET方法傳送，傳送目標可以是虛擬的
・只寫<form>到</form>的範圍
・在所有元素中包含name、id和class屬性
・name和id的值可以用虛擬值補充。class值可以留空
・文字輸入元素中要加入placeholder元素來補充值
・複選框和單選按鈕的選項要加入label元素(for屬性)

#組成要求
電子郵件：
・單行輸入，最多200字，必填
姓名：
・單行輸入，最多20字，選填
電話號碼：
・單行輸入，最多20字
生日：
・月份：下拉選單(1月至12月)
・日期：下拉選單(1日至31日)
性別：
・單選按鈕(男性、女性、其他)
職業：
・單選按鈕(畢業生、新人、學生)
希望內容：
・複選框("資料請求"、"參觀拜訪"、"其他")

#條件限制：
・只寫<table>到</table>的範圍
・在子元素中包含id、class和屬性

・組成要求中空白的欄位可以保持空白
