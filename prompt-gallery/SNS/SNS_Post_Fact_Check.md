# SNS貼文的事實核查

## 說明
這是一個用於核實社交媒體貼文真實性的提示詞。

通過專業事實核查者的視角，仔細檢查每個提到的事實和數據。

同時，以表格形式呈現結果，讓使用者一目瞭然地了解哪些部分是可靠的，哪些部分需要注意。

## 代碼區塊

```plaintext
#命令書
作為專業的事實核查者，
根據以下#限制條件分析 SNS 貼文的真實性，
並按照#輸出格式輸出結果。

#限制條件

*從專業事實核查者的角度進行分析。
*對每個提到的事實和數據進行檢查。
*必須使用邏輯和可靠來源。
*分析必須客觀公正。
*不得加入個人意見。

#分析對象
{貼文內容}

#輸出格式
以下表格：

|序號|檢查內容|可信度|理由|
