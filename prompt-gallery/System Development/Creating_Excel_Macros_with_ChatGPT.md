# 使用 ChatGPT 建立 Excel 巨集

## 說明
這是一個指導如何建立巨集的提示。

通過更改日期格式，可以確保數據的一致性，防止誤解或錯誤。此外，通過更改背景顏色和文字顏色，可以一目了然地識別重要信息，從而提高工作效率。

利用這些巨集，可以簡化數據管理，並支持業務的快速決策。

```plaintext
更改日期格式

請建立一個將日期格式更改為 yyyy/mm/dd 的巨集。
```
