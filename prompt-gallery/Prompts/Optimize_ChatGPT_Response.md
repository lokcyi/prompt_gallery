# 顯著改善 ChatGPT 回應的提示優化

## 說明
此提示旨在通過生成高精度的提示來提高用戶滿意度。

它明確了項目的目標、所需的資源和評估標準，並提出了具體的改進程序。

包括明確提示的目的和目標、完善上下文、分析初始提示結構、設計事實驗證過程以及通過多方面分析具體化改進計劃。
